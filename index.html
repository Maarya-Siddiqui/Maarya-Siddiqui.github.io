This website is launched using GitHub Pages. I began by creating a repo named after my GitHub username, ending with .io. Next, I cloned the repo, and installed GitHub Desktop to commit and push local changes to the website! It was a simple, and straightforward process. I had some trouble with seeing local changes take effect on the website. However, I consulted some resources and learned that local commits take time to take effect. Overall, really enjoyed this process and will be using GitHub Pages more often for personal web development projects!
I built this page to house all my labs from CPS530. Links to all my webpages can be found above. Internally, they are stored in an unordered list as hyperlinks, leading to new pages. Enjoy!
